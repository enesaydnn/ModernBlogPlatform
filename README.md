# 🚀 ModernBlogPlatform

ModernBlogPlatform is a modern full-stack blogging platform developed with **PHP** and **MySQL**. It provides a clean reading experience while offering an intuitive admin panel for managing articles, categories, and users. The project focuses on responsive design, secure content management, and scalable architecture.

## ✨ Features

- 📝 Blog & Article Management
- 👤 User Authentication
- 🔒 Admin Dashboard
- 📂 Category Management
- 🔍 Search Functionality
- 📱 Responsive Design
- ⚡ Fast & Lightweight
- 🤖 AI-Ready Architecture

## 🛠 Tech Stack

- PHP
- MySQL
- HTML5
- CSS3
- Bootstrap
- JavaScript

## 🚀 Getting Started

```bash
git clone https://github.com/enesaydnn/ModernBlogPlatform.git
```

Configure your database, update the connection settings, and run the project on a local PHP server.
